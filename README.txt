Το project bbska αφορά σε μια διαδικτυακή εφαρμογή που σκοπό έχει τη διαχείριση πληροφοριών
με έναν τρόπο που θυμίζει τις ξύλινες ρωσικές κούλες γνωστές ως μπάμπουσκες ή ματριόσκες.
Ο χρήστης μπορεί να δημιουργεί κόμβους πληροφορίας και μέσα σε κάθε κόμβο πληροφορίας να
εντάσσει άλλους κόμβους πληροφορίας κοκ. Οι κόμβοι πληροφορίας ονομάζονται και μονάδες
πληροφορίας, ή απλώς μονάδες, ή κόμβοι.

Κάθε μονάδα πληροφορίας αποτελείται από ένα ID και μια σειρά από key/value pairs, ενώ
παρέχονται κάποια υποτυπώδη μέσα διαμόρφωσης που αφορούν στην εμφάνιση των δεδομένων.
Οι μονάδες μπορούν να είναι ιδιωτικές ή δημόσιες, όπου τις ιδιωτικές μονάδες μπορεί να
τις προσπελαύνει μόνο ο ιδιοκτήτης, ενώ οι δημόσιες μονάδες είναι προσπελάσιμες από
οποιονδήποτε. Δικαίωμα εισαγωγής, τροποποίησης και διαγραφής έχει, φυσικά, μόνον
ο ιδιοκτήτης ασχέτως ιδιωτικότητας της μονάδας.

Στην καρδιά του προγράμματος βρίσκεται μια σχεσιακή database με ελάχιστο αριθμό πινάκων.
Όταν ο χρήστης ζητά κάποια πληροφοριακή μονάδα, τότε εκτελείται μεγάλος αριθμός (μικρών)
queries προκειμένου να σχηματιστεί και να αποσταλεί η πλήρης εικόνα του ζητούμενου κόμβου.
Η όλη διαδικασία είναι εξαντλητική μέχρι τους κόμβους τελευταίου επιπέδου. Για να αποφευχθεί
η διαρκής καταπόνηση του server, τα αποτελέσματα κασάρονται σε files και όσο δεν γίνονται
αλλαγές αποφεύγεται η εκτέλεση των ιδίων queries εφόσον ζητηθεί κόμβος του οποίου η εικόνα
έχει κασαριστεί. Όποτε ο χρήστης κάνει αλλαγές, τα κασαρισμένα στοιχεία διαγράφονται και
δημιουργούνται εκ νέου με τις πρώτες κρούσεις. Ο χρήστης μπορεί, επίσης, να διαγράψει τα
cache files όποτε θέλει (reCache).
